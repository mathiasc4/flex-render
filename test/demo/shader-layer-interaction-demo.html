Interaction legend

  • Pointer: cyan marker when no button is pressed.
  • Primary button: red pointer marker and red button dot.
  • Secondary button: magenta pointer marker and magenta button dot.
  • Auxiliary button: green pointer marker and green button dot.
  • Last click: yellow/orange ring. Repeated clicks alternate color through clickSerial.
  • Active drag: cyan line/rectangle, green start marker, magenta current marker.
  • Completed drag: persistent line/rings. Repeated completed drags alternate color through dragSerial.

Instructions

  1. Move the pointer over the viewer.
  2. Hold left, right, or middle mouse buttons.
  3. Click repeatedly at the same position to verify clickSerial.
  4. Drag and release to verify drag start/current/end state.
  5. Toggle interaction forwarding off; all shader-visible interaction state should clear.
  6. Toggle notifyOnMove to compare event-driven and polled readouts.
  7. Reorder the interaction-debug layer above/below other layers to test composition.

The demo readouts use only FlexDrawer/FlexRenderer APIs. They do not install independent pointer handlers.
